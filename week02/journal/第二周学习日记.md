# 对正则表达式的补充

- 元字符（特殊字符）

| 字符    | 含义                                |
| :------ | :---------------------------------- |
| `.`     | 匹配除换行符外的任意单个字符        |
| `^`     | 匹配字符串开头                      |
| `$`     | 匹配字符串结尾                      |
| `*`     | 匹配前一个字符0次或多次             |
| `+`     | 匹配前一个字符1次或多次             |
| `?`     | 匹配前一个字符0次或1次              |
| `{n}`   | 匹配前一个字符恰好n次               |
| `{n,}`  | 匹配前一个字符至少n次               |
| `{n,m}` | 匹配前一个字符n到m次                |
| `[ ]`   | 字符组，匹配其中任意一个字符        |
| `( )`   | 分组，捕获匹配内容                  |
| `|`     | 逻辑或，匹配左边或右边              |
| `\`     | 转义字符，将元字符转义为普通字符    |
| `-`     | 连字符，在字符组中表示范围（如a-z） |

-  预定义字符类（需要转义）

| 字符 | 含义                           |
| :--- | :----------------------------- |
| `\d` | 数字，等价于[0-9]              |
| `\D` | 非数字                         |
| `\w` | 单词字符（字母、数字、下划线） |
| `\W` | 非单词字符                     |
| `\s` | 空白字符（空格、制表符等）     |
| `\S` | 非空白字符                     |
| `\b` | 单词边界                       |
| `\B` | 非单词边界                     |

-  锚点与断言

| 字符      | 含义                     |
| :-------- | :----------------------- |
| `\A`      | 字符串绝对开头           |
| `\z`      | 字符串绝对结尾           |
| `\Z`      | 字符串结尾（允许换行前） |
| `(?=...)` | 正向先行断言             |
| `(?!...)` | 负向先行断言             |

# RESTFUL API

## 基础语法(选择动作)

- GET查询资源
  - `GET /users` --查询所有用户
  - `GET /users/123` --查询id为123的用户
- PUT 完整更新资源
  - `PUT /users/123` --完整更新用户123
- PATCH 部分更新资源
  - `PATCH /users/123` --只更新123的部分字段，通常用于更精细的操作
- POST 创建资源
  - `POST /users` --创建新用户
- DELETE 删除资源
  - `DELETE /users/123` --删除用户123

## 添加查询条件

- 分页
  - `/users?page=2&limit=10`(查询第二页，每页十条信息)
- 过滤
  - `/users?gender=male&age=25`(查询性别为男，年龄25的用户)
- 排序
  - `/users?sort=created_at&order=desc`(按创建时间逆序排序用户)

# 对于mybatis项目更改spring框架的理解
